github備忘録

①.初期設定

1.gitのインストール
 homeディレクトリで
 $ sudo apt-get install git

2.githubアカウントの作成
 "https://github.com/”でアカウントを作る。

3.SSHkeyの設定
 SSHとは、Secure Shell（安全なシェル）の略で、暗号や認証の技術を利用して、安全にリモートコンピュータと通信するためのプロトコルのことである。パスワードなどの認証部分を含むすべてのネットワーク上の通信が暗号化される（Wikipediaより）
 今回は、まずSSHkeyという鍵を作る（この鍵は公開鍵と秘密鍵でついになっている。）。次に、githubに公開鍵を渡して、秘密鍵は自分のPCに持っておく。そうすることで、他の誰かが自分のアカウントを使って勝手にソースを上げることを防ぐことができる。
 
○SSHkeyの作成
ターミナルのhomeディレクトリ上で、以下のコマンドを実行する。
　ssh-keygen -t rsa -C "アカウント作成時に登録したメアド"
すると、homeの/.sshに秘密鍵（id_rsa）と公開鍵(id_rsa.pub)が作られる。

○SSHkeyの登録
・"https://github.com/”一番右上にある下矢印をクリック
・下から2番目のSettingsをクリック
・左の一覧にある"SSH and GPG keys"をクリック
・New SSH keyをクリック
・Titleは何でもOK。Keyの部分に、先ほど作成した公開鍵をコピペする。

4.Githubにディレクトリを上げる
○レポジトリの作成
・"https://github.com/"右上の＋ボタンをクリック
・New repositoryをクリック
・Repository nameに好きな名前を入力して、Create repositoryをクリック
・一番上に出てくる"https://github.com/"から始まるURLをコピー

○作成したレポジトリにアップロード
・Githubに上げたいディレクトリの中で以下のコマンドを実行する。
 git init
 git add *
 git commit -m "コメント"
 git remote add origin 先ほど作ったレポジトリのURL
 git push -u origin master

※Please tell me who you are.というエラーができたら、ターミナルのhomeディレクトリで以下のコマンドを実行する。
git config --global user.email 登録した時のメアド

以上。
